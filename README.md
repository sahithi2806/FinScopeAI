# FinScope AI
An AI-powered stock market analysis platform with trend detection, volatility analysis, and visualization.

## Features
- Fetches real-time stock data
- Visualizes price trends
- Calculates moving average
- Detects trend 
- Computes percentage change
- Measures volatility 
- Generates insights

## Tech Stack
- Python
- yfinance
- matplotlib

## Future Scope
- AI-based explanations
- News sentiment analysis
